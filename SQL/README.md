- 1986 padrão ANSI
- 1987 padrão ISO

- Sistemas
  - MySQL
  - SQL Server (T-SQL)
  - Oracle (PL/SQL)
  - Access (JET SQL)
  - PostgreSQL
  - Sybase
  - DB2

- Visões, Stored Procedures e Funções

# Grupos de Comandos

- `DDL` (Data Definition Language)
  - CREATE
  - ALTER
  - DROP
- `DML` (Data Manipulation Language)
  - INSERT
  - UPDATE
  - DELETE
- `DCL` (Data Control Language)
  - GRANT : permissão
  - REVOKE : retira permissao
- `DQL` (Data Query Language)
  - SELECT
