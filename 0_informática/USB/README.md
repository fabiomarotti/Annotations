### Lançamentos
- USB 1.0 (1996)
- USB 1.1 (1998) : de 1,5 Mbps até 12 Mbps
- USB 2.0 (2000) : 480 Mbps 
- USB 3.0 (2009) :   5 Gbps 
- USB 3.1 (2017) :  10 Gbps
- USB 3.2 (2019) :  20 Gbps
- USB 4.0 (xxxx) :  40 Gbps

### Informações
- USB 2.0:
  - possui 4 pinos   
- USB 3.0:
  - possui 9 pinos 
  - fornecimento de energia 80% mais eficiente
- USB 4.0 
  - Compativel com Thunderbolt 3
